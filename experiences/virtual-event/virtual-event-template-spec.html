虛擬事件模板規範

{{ page.description }}

視訊與音訊

以下列出視訊和音訊建議。

視訊

格式 MP4
編解碼器 264 公路
比特率 最高轉譯的位元率的 2 倍
影格速率 常數，通常介於每秒 15 至 60 幀
長寬比 您的視訊外觀比例將會保留。16:9 的外觀比例是常見的。

音訊

編解碼器 AAC (進階音訊轉碼器)
比特率 常數，每秒 128 公里
頻道數 立體聲

有關錄製，導出，上傳和發布視頻以確保最佳播放的最佳做法的建議，請參閱最佳做法：視頻創建，轉碼和上傳工作流程。

影像

虛擬事件範本中會使用各種影像。圖片建議如下所示。

背景影像

影像大小取決於客戶嘗試支援的中斷點，通常不是 1920x1080 的作品，但影像應該進行最佳化，因此不是巨大的，以獲得最佳效能。

標頭圖誌

標題標誌圖像將按比例縮小到 178 像素寬。

議程

默認圖像大小是 320 像素 x 180 像素。圖像將被裁剪為此大小。

揚聲器

揚聲器圖像應具有 200 像素的最大寬度。影像不會重新調整大小。默認情況下，模板 CSS 配置揚聲器圖像與 50％ 的邊框半徑，這將使圖像圓形。揚聲器下方顯示的揚聲器簡介僅限於大約 3 行 15 個字。

贊助商

贊助商圖像將被調整為100px寬，並且不透明度設置為80％。CSS樣本提供以覆蓋此行為。

字型

該模板經過優化，可與濃縮的Roboto字體。可以使用風格 Gallery界面內部的菜單。其他字體可以使用，但可能需要變更 CSS 檔案。有關使用自定義字體的信息，請參見在門戶體驗中使用自定義字體。

議程

活動議程列出了現場活動和其他會議的日程安排。創建並發布議程後，它將顯示在活動主頁上。有關創建活動議程的信息，請參閱管理活動議程 .
