將配置文件添加到體驗

{{ page.description }}

虛擬事件體驗需要設定檔案包含在站台中。配置文件包括網站設置，佈局選項，事件時間表，揚聲器和其他將顯示的內容。建立新的虛擬事件體驗之後，新增設定檔案是應該完成的第一件事

若要將組態檔新增至虛擬事件體驗，請依照下列步驟執行。

  1. 轉到以下鏈接：
    https://virtual-events-experience.s3.amazonaws.com/v1/config.txt
  2. 將內容複製到剪貼簿。
  3. 開啟「 圖庫」 模組。
  4. 編輯體驗。
  5. 點擊外觀和行為>頁眉和頁腳在左側導航欄中。
  6. 將配置文件粘貼到自定義標題HTML場地。
  7. 點擊保存。

確保你重新發布經驗所以變化出現。

更新政策密鑰和資產基礎網址價值觀

為了讓倒數計時器在模擬視頻上顯示，政策密鑰需要使用啟用搜索的策略鍵更新配置文件中的值。提供的值不適用於您的帳戶。如果無法執行此動作，則影片不會顯示倒數計時。有關獲取啟用搜索的策略密鑰的步驟，請參閱開發人員文檔代碼樣例：生成策略密鑰。創建新的策略密鑰後，請更新policyKey配置文件中的值。

這資產基礎網址值路徑用於加載模板使用的文件。在大多數情況下，這可以單獨保留，除非正在執行高度的自訂。那裡的路徑值應該被刪除。

之後政策密鑰和資產基礎網址配置文件中的值已更新，重新發布體驗。
