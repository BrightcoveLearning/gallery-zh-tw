自定義虛擬事件體驗的外觀和行為

{{ page.description }}

要配置外觀和行為設置，請編輯體驗並單擊外觀與行為在左側導航欄中。您可以使用下列設定：

  • 頁眉和頁腳 -為體驗配置自定義頁眉/頁腳
  • 風俗 -使用自定義CSS和JavaScript文件
  • 背景圖像和視頻 -配置背景圖片或視頻

設定自訂頁頭/頁尾HTML

圖庫提供使用網站自訂頁首和/或頁尾的選項。這提供了自定義虛擬活動體驗的能力，使其更符合您公司網站的外觀和感覺。您也可以在帳戶層級設定自訂頁首/頁尾。有關在帳戶級別配置自定義頁眉/頁腳的信息，請參見配置圖庫設置。

要將虛擬事件體驗配置為使用自定義頁眉或頁腳，請按照以下步驟操作。

  1. 設計頁眉/頁腳用於您的網站。頁眉/頁腳可以包含任何有效的 HTML，JavaScript 或 CSS 代碼。
  2. 點擊外觀和行為>頁眉和頁腳在左側導航欄中。
  3. 將標頭HTML粘貼到自定義標題HTML場地。
  4. 選擇在自定義標題下包含默認標題選項，以在自定義標題下方顯示默認模板標題。
  5. 將頁腳HTML粘貼到自定義頁腳HTML場地。
  6. 選擇在自定義頁腳下包括默認頁腳選項可在自定義頁腳下方顯示默認模板頁腳。
  7. 點擊保存。

使用自訂頁首和頁尾的注意事項

  • 圖庫不會對頁眉/頁腳提供的 HTML 進行任何驗證
  • 頁眉和頁腳 HTML 沒有明確的字符限制
  • 支援外部 CSS 和 JavaScript 檔案
  • 代碼應該遵循響應式設計原則，以便在設備之間正確擴展
  • 當網站使用引導程序時，我們已經看到了 CSS 衝突（圖庫也使用引導程序）; 確保您檢查並解決任何衝突

設定自訂設定

自訂設定提供自訂網站外觀和風格的功能。要配置自定義設置，請單擊外觀和行為>自定義在左側導航欄中。

您可以使用下列設定：

  • 自定義CSS URL -用於指定自定義.css文件的位置以自定義樣式。點擊查看CSS鏈接以查看當前用於設置網站樣式的CSS文件。
  • 自定義JavaScript URL -用於指定自定義.js文件的位置，以自定義網站的樣式和/或行為。
  • 自定義圖標網址 -用於指定網站圖標的位置。提供圖示支援的瀏覽器通常會在頁面的瀏覽器索引標籤上顯示圖示，以及書籤清單中頁面名稱旁邊。
  • 自定義移動應用程序圖標 -用於選擇將站點保存到設備主屏幕時用作主屏幕圖標的圖像。Brightcove 建議您上傳高解析度 (我們建議您上傳 1024 像素 x 1024 像素)、方形影像和圖庫將縮放影像，並使用最適合您裝置的版本。
  • 自定義響應標題 -輸入HTTP標頭，該標頭將響應對門戶的請求而發送。例如，可能會新增標頭來新增內容安全性原則 (CSP) 或 HTTP 嚴格傳輸安全性 (HSTS)。

配置背景圖片或視頻

圖庫提供了顯示站點自定義背景圖像或視頻的選項。要配置背景圖片或視頻，請點擊外觀和行為>背景圖像和視頻在左側導航欄中。

添加背景圖片

  1. 若要指定影像，您可以：
    • 從檔案系統拖曳影像，然後將其放置到放置區域
    • 點擊瀏覽然後從文件系統中選擇一個圖像
    • 輸入遠端影像檔案的 URL
  2. 點擊保存。

點擊去掉刪除圖像。

添加背景視頻

視頻雲視頻也可以用作背景視頻。將視頻雲視頻ID粘貼到影片編號字段，然後單擊保存。

筆記：如果視頻 ID 無效，將顯示上傳的背景圖片或默認背景圖片。
