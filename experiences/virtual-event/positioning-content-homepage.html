在首頁上定位內容

{{ page.description }}

配置文件中的幾個對象具有參數，這些參數使您可以指定對象應在頁面上顯示的位置。例如，靜態內容，發言人，曲目和讚助者都可以在頁面上重新定位。可以定位的對像在配置文件中將具有以下屬性：

  • insert_target -CSS選擇器（簡單的類或ID），將用作呈現給定對象的起點
  • insert_mode -如何插入內容
    • after
    • before
    • append
    • prepend

確定目標

這insert_target參數需要一個CSS選擇器（使用簡單的類或ID格式）。與該選擇器匹配的元素將用作渲染給定對象的起點。確定要使用的CSS選擇器或ID的最佳方法是瀏覽到虛擬事件主頁，然後使用瀏覽器開發人員工具檢查頁面中的元素。

您會注意到，主頁的主要部分是使用<div>標籤。

選擇插入模式

確定要在層次結構中的哪個位置插入對像後，insert_mode屬性將確定如何將對象插入層次結構。

  • after -將對象放在指定的位置之後div在同一水平
  • before -將對象放在指定的對象之前div在同一水平
  • append -將對象放入指定的對象div作為最後一個孩子
  • prepend -將對象放入指定的對象div作為第一個孩子

下圖展示了各種效果insert_mode價值觀。

對象定位示例

下面的代碼示例演示如何使用靜態內容對像在輔助集合上方插入標題。


{
    "insert_target": ".bc-on-demand-section",
    "insert_mode": "prepend",
    "content": "<h2 class='bc-on-demand-section-title'>ON-DEMAND SESIONS</h2>",
    "require_roles": [],
    "omit_roles": []
}

使用瀏覽器工具，您可以看到h2作為第一個元素插入bc-on-demand-section div。
