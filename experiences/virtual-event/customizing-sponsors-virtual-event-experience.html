定制活動贊助商

{{ page.description }}

贊助商圖像可以添加到虛擬事件主頁。

贊助者信息是通過編輯sponsors配置文件中的對象。

  • name -贊助商名稱
  • image_url -贊助商徽標，圖片將調整為100px寬
  • link_url -圖片的目標網址

這sponsors配置文件中的object對像還具有用於控制贊助者信息在首頁中顯示位置的參數。有關設置贊助商部分的信息，請參見在首頁上定位內容。

去除不透明性並調整徽標圖像的大小

默認情況下，徽標圖像的不透明度為80％。以下CSS樣式可用於將不透明度設置為100％。這最大寬度屬性可用於更改圖像的大小。


<style>
.bc-sponsors-section .bc-sponsors-list img {
    -webkit-filter: none;
    opacity: 1;
    max-width: 200;
}
</style>
編輯體驗，然後將CSS添加到頂部自定義標題HTML。場地。
