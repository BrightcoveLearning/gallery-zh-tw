處理靜態內容

{{ page.description }}

靜態內容對象用於將自定義 HTML 注入模板。這個樣本英雄橫幅是使用文字和圖像的組合構成，並注入頁面頂部。

靜態內容可以是任何 HTML，並且可以在模板中的任何位置注入。靜態內容對像已添加到static_content配置文件中的對象。有關如何在頁面上放置靜態內容的信息，請參見在首頁上定位內容。
