自定義活動發言人

在本主題中，您將學習如何自訂出現在虛擬活動體驗首頁上的活動揚聲器。

您可以在虛擬活動首頁加入演講者影像和簡短的簡報。

通過編輯speakers配置文件中的對象。

  • name -演講者姓名
  • description -在其圖像下方顯示的發言人信息。這僅限於大約 3 行 15 個字。
  • image_url -演講者形象。建議 200 像素的最大寬度。影像不會重新調整大小
  • link_url -（可選）圖片的目標網址（例如，可以鏈接到演講者的LinkedIn頁面）

默認情況下，模板CSS將揚聲器圖像配置為border-radius 50％將使圖像變為圓形。

這speakers配置文件中的對像還具有控制發言者信息在首頁中顯示位置的參數。有關放置揚聲器部分的信息，請參閱在首頁上定位內容。
