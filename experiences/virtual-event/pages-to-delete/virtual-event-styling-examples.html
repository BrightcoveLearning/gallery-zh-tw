虛擬事件樣式示例

本主題提供了一些示例代碼，可用於自定義虛擬事件模板。

社交分享圖標

默認情況下，如果視頻共享設置已啟用體驗，它們將不會顯示在視頻詳細信息頁面上的視頻旁邊。以下CSS將在視頻詳細信息頁面上顯示社交圖標。


.share-list {
  display: block !important;
}
.share-list a {
  display: inline-block !important;
}
.share-list li {
  display: block !important;
}
