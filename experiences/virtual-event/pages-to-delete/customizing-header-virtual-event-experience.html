自定義虛擬事件體驗的標題

在本主題中，您將學習如何自定義“虛擬事件體驗"頁面的標題。

標題將出現在所有頁面的頂部。標頭是從組態檔案中的值建構的。標頭是由下列元件組成：

  • 標頭標誌
  • 連結
  • 社交圖示
  • 搜尋

標頭標誌

標題圖片由images配置文件中的對象。

  • header_logo -圖片的網址顯示在標題中

支持.Png和.jpg文件。標頭徽標將按比例縮小到178px寬。

連結

默認情況下，家，現場直播和按需會議鏈接硬編碼在模板中：

  • 家 -回到首頁，頁面頂部
  • 現場直播 -轉到精選集（頂級視頻播放器）
  • 按需會議 -轉到曲目部分

可以通過編輯其他鏈接來添加其他鏈接header_links配置文件中的對象。

  • link_url -目標網址
  • label -標籤出現在標題中

社交圖示

默認情況下，標題將顯示社交圖標：

  • Facebook
  • Instagram
  • 推特
  • LinkedIn
  • YouTube
標題中顯示的社交圖標可以通過編輯social配置文件中的對象。有關信息，請參閱配置虛擬事件體驗的社交圖標。

搜尋

要啟用搜索並顯示搜索字段，啟用搜索體驗。默認情況下，標題中的搜索字段將顯示一行用於輸入文本。

示例中的搜索框已設置樣式，以在搜索字段周圍顯示一個框。

標頭特定的樣式

移動設備上的鏈接

默認情況下，使用移動設備時，不會顯示添加到標題的鏈接。以下JavaScript解決了該問題。


for (let i = 0, l = ciabConfig.header_links.length; i < l; i++) {
  let link = ciabConfig.header_links[i];
  let li = document.createElement('li');
  let ul = document.getElementById('main-nav-cat');
  li.className = 'main-nav-category main-nav-category-home';
  li.innerHTML = ('' + link.label + ''); ul.appendChild(li);
}

格式化搜索字段

默認情況下，搜索字段將只顯示一行用於輸入文本。以下CSS將搜索字段格式化為文本框。

    margin-right: 45px;
    width: 170px;
}

	color: #fff;
	margin-top: -10px;

	font-size: 14px;
	font-weight: 500;
	line-height: .88;
	text-transform: uppercase;
	display: block;
    padding: 2px 5px 2px 30px;
	width: inherit;
	border: 1px solid #fff;
}
