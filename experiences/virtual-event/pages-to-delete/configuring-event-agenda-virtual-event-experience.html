配置事件議程

在本主題中，您將學習如何自定義虛擬事件體驗的事件議程。

事件議程通常用於顯示出現在精選集合中的預定實時事件。與每個工作階段一起顯示的按鈕取決於工作階段的狀態：

  • 活動前 -沒有顯示按鈕，顯示倒數計時器
  • 中 -活動進行時立即直播
  • 賽后 -按鈕將顯示過去事件的VIEW SESSION。

根據議程中的天數，表演將出現一個下拉菜單，使觀看者可以選擇當天的時間表進行觀看。

通過編輯schedule配置文件中的對象。

  • name -會話名稱
  • description -會話描述-當前未顯示在頁面上，但可以使用自定義JavaScript顯示
  • starts_at -會話開始日期和時間
  • ends_at -會話結束日期和時間
  • live_id - 不曾用過
  • vod_id -影片ID
  • speakers -演講嘉賓信息
  • image_url -會話圖像。默認圖像大小是 320 像素 x 180 像素。圖像將被裁剪為此大小。

這schedule配置文件中的對像還具有用於title出現在時間表上方，並設置時間表是否顯示在expanded模式。
