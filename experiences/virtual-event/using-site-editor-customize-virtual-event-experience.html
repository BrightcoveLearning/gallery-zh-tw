使用站點編輯器自定義虛擬活動體驗

{{ page.description }}

圖庫網站編輯器用於自定義體驗的外觀。按一下體驗標題以開啟網站編輯器。「網站編輯器」提供下列功能：

  • 將對象添加到頁面
  • 將視頻添加到網站
  • 自定義網站樣式
  • 配置號召性用語
  • 自定義外觀和行為
  • 自定義網站功能
  • 自定義視頻和播放
  • 自定義站點配置
  • 配置站點詳細信息

按一下左側導覽中的功能表即可存取這些選項。

將物件新增至頁面

可以將諸如文本，圖像和鏈接之類的對象添加到頁面中，以使您可以自定義視頻品牌並為其打上烙印。要了解如何向頁面添加對象，請參見自定義虛擬事件體驗頁面的內容。

將視訊新增至網站

要將視頻添加到站點，請單擊影片在左側導航欄中。有關如何將視頻添加到視頻站點的完整詳細信息，請參見將視頻添加到虛擬事件體驗。

自訂網站樣式

要自定義網站上使用的顏色和字體，或選擇其他主題，請單擊風格在左側導航欄中。有關如何自定義樣式的完整詳細信息，請參見自定義門戶體驗的風格。

設定動作呼叫

動作呼叫提供在頁面上包含自訂 HTML、影像或廣告的功能。並非所有範本都支援行動呼叫。要配置號召性用語，請點擊行動呼籲在左側導航欄中。

有關配置號召性用語的完整詳細信息，請參見在門戶網站體驗上配置號召性用語。

自訂外觀和行為

外觀和行為設置用於配置自定義頁眉/頁腳，使用自定義CSS和JavaScript文件以及配置背景圖片或視頻。要配置外觀和行為設置，請單擊外觀與行為在左側導航欄中。

有關外觀和行為設置的完整詳細信息，請參見自定義虛擬事件體驗的外觀和行為。

自訂網站功能

網站功能設定可用來啟用網站搜尋、設定社交分享選項，以及允許對影片留言。要配置站點功能，請單擊網站功能在左側導航欄中。

有關如何自定義網站功能的完整詳細信息，請參閱自定義門戶體驗的站點功能。

自訂視訊和播放設定

視訊和播放設定可讓您選取體驗的自訂播放器、設定受眾整合和潛在客戶表單，以及設定視訊選項，例如下載和相關連結。要自定義視頻和播放設置，請單擊視頻和播放在左側導航欄中。

有關配置視頻和播放設置的完整詳細信息，請參閱定制門戶體驗的視頻和播放設置。

自訂網站組態

站台組態設定可讓您在自訂網域中託管站台、使用 SSL 保護網站，或限制對網站的存取。要配置站點配置，請單擊現場配置在左側導航欄中。

有關站點配置設置的完整詳細信息，請參閱定制門戶網站體驗的站點配置。

設定網站詳細資料

概觀設定可讓您編輯網站的名稱、描述和語言。要配置站點詳細信息，請單擊概述在左側導航欄中。

有關配置站點詳細信息的完整詳細信息，請參閱配置門戶體驗的詳細信息。
