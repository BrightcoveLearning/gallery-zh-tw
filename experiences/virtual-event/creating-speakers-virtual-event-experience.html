創建事件揚聲器文件

{{ page.description }}

您可能已經在第三方工具（例如 Microsoft Excel 或 Google 表格）中創建了活動發言人列表。議程模塊提供上傳活動發言人的 .csv 文件的功能。上載演講者文件後，可以使用議程模塊編輯演講者信息。

筆記：可以在此處下載示例揚聲器模板文件。

列命名要求

創建演講者文件時，您可以下載示例演講者模板並對其進行編輯或使用您可能已創建的現有演講者文件。Agenda 模塊查找特定列名稱的存在。這些列將映射到適當的揚聲器字段。導入過程中將忽略揚聲器文件中可能存在的其他列。為了上傳演講者文件，需要以下列：

  • first_name -演講者的名字
  • last_name -演講者的姓氏

以下列是可選的，但如果它們存在於您的揚聲器文件中，它們也將被導入：

  • id - 不能由用戶定義。如果用戶將演講者列表導出為 CSV，則此 ID 將被填充。這用於更新數據庫中的現有條目。
  • email -演講者的電子郵件地址
  • title -演講嘉賓
  • company -演講者公司
  • bio -演講者簡介
  • image - 揚聲器圖像

當您準備好將揚聲器導入 Agenda 模塊時，請將其導出為 .csv 文件。議程模塊只能導入 .csv 文件。

筆記：有關如何將議程文件上傳到議程模塊的信息，請參閱上傳和自定義活動議程.
