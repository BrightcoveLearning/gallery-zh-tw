配置伴隨組件以實現頁內體驗

{{ page.description }}

所有頁面內體驗範本都提供具備隨附元件的能力。輔助元件包括：

  • 自訂 HTML
  • 影像
  • 文字

使用隨附的元件，播放器外部的內容可以根據播放的視訊而變更。例如，播放產品影片時，播放器右邊可能會有一個可點選的影像，可以開啟該產品的詳細資料。伴隨組件類似於卡和鏈接互動，差異在於，卡和鏈接顯示在視頻播放器本身。視頻在播放器中時，隨播組件會顯示在視頻播放器的右側玩遊戲狀態。

筆記：有關添加鏈接和卡片交互的信息，請參閱為頁內體驗添加交互.
筆記：對體驗進行任何更改後，請確保您重新發布經驗因此更改會出現在顯示體驗的網站上。

檢視隨附元件

要查看已配置的配套組件，請編輯體驗，然後單擊“確定"。影片在左側導航欄中。已配置互動的視頻將在互動性指示互動類型的列。

伴隨組件包括 HTML ( )、影像 ( ) 和文字 ( ）。

筆記：關聯 （ ) 和卡片 ( ) 圖示可能會出現，表示視訊已設定連結和卡片。有關鏈接和卡片的信息，請參閱為頁內體驗添加交互.

設定隨附的 HTML 元件

若要設定隨附的 HTML 元件，請依照下列步驟執行：

  1. 編輯體驗，然後單擊影片在左側導航欄中。
  2. 按一下影片以選取它。
  3. 點擊添加交互性>配套 HTML或點擊HTML () 圖示。互動編輯器將會開啟。
  4. 進入一個名稱用於HTML組件。
  5. 粘貼你的 HTML 代碼。
  6. 要配置組件的開始時間和持續時間，請點擊配置關聯。
  7. 選擇一個開始時間和持續時間：
    • 顯示整個視頻 -該組件將出現在整個視頻中
    • 選擇開始時間和持續時間 -您將手動設置組件的開始時間和持續時間
  8. 設置期間對於組件。
    筆記：您也可以使用播放器下方的滑桿來設定持續時間。將左箭頭拖動到組件應該出現的時間，然後將右箭頭拖動到組件應該消失的時間。
  9. 查看直到視頻結束如果您只想設置開始時間並在視頻結束之前顯示該組件。
  10. 查看將此互動應用於版式中的所有視頻如果您希望此組件出現在體驗中的所有視頻中。
  11. 點擊保存保存更改。
筆記：要將交互分配給另一個視頻，請使用播放器上方的視頻選擇器下拉列表選擇另一個視頻。

設定隨附影像元件

若要設定隨附的映像元件，請依照下列步驟執行：

  1. 編輯體驗，然後單擊影片在左側導航欄中。
  2. 按一下影片以選取它。
  3. 點擊添加交互性>配套圖片或點擊圖片 （) 圖示。互動編輯器將會開啟。
  4. 進入一個名稱用於圖像組件。
  5. 若要指定影像，您可以：
    • 從檔案系統拖曳影像，然後將其放置到放置區域
    • 點擊瀏覽然後從文件系統中選擇一個圖像
    • 輸入遠端影像檔案的 URL
  6. 設置圖像尺寸。
    • 圖片將調整為容器大小 -尺寸大於其所在容器的圖像將按比例縮小以適合。較小的影像不會升頻。
    • 寬度和高度 -手動輸入圖像的寬度和高度
  7. 設置圖像結盟。
  8. 進入一個點擊後到達網址。這是點擊圖像時要轉到的 URL。你也可以在新窗口中打開鏈接。
  9. 要配置組件的開始時間和持續時間，請點擊配置關聯。
  10. 選擇一個開始時間和持續時間：
    • 顯示整個視頻 -該組件將出現在整個視頻中
    • 選擇開始時間和持續時間 -您將手動設置組件的開始時間和持續時間
  11. 設置期間對於組件。
    筆記：您也可以使用播放器下方的滑桿來設定持續時間。將左箭頭拖動到組件應該出現的時間，然後將右箭頭拖動到組件應該消失的時間。
  12. 查看直到視頻結束如果您只想設置開始時間並在視頻結束之前顯示該組件。
  13. 查看將此互動應用於版式中的所有視頻如果您希望此組件出現在體驗中的所有視頻中。
  14. 點擊保存保存更改。
筆記：要將交互分配給另一個視頻，請使用播放器上方的視頻選擇器下拉列表選擇另一個視頻。

編輯/刪除配件元件

將鼠標懸停在互動性列將顯示該類型的所有互動。在此範例中，已設定映像元件。

要編輯組件，請將鼠標懸停在互動性列並單擊編輯（) 組件旁邊的圖標噸或單擊視頻標題。互動編輯器將會開啟。編輯組件，然後保存變化。

要刪除組件，請將鼠標懸停在互動性列並單擊刪除（) 圖標旁邊的組件。

也可以通過單擊編輯圖標 (） 在行動柱子。這將顯示視頻的所有互動列表。

按一下編輯圖示以編輯組件，或按一下刪除圖示以刪除互動。

伴隨組件也可以從佈局菜單。選擇玩遊戲狀態和伴隨圖標將顯示在體驗上。
