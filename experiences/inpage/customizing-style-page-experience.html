    自訂頁面內體驗的風格

    在本主題中，您將學習如何自訂頁面內體驗的樣式。

    樣式控製字體，字體大小和顏色頁內體驗。要自定義樣式，請編輯體驗，然後單擊風格在左側導航欄中。

    注意：對頁內體驗進行任何更改後，請確保您重新發布體驗因此更改會顯示在顯示體驗的網站上。

    切換裝置狀態

    頁面頂端的連結可用來自訂不同裝置狀態的樣式，包括桌上型電腦、平板電腦或行動裝置。

    默認情況下，桌面自定義樣式時顯示模式。裝置狀態可用來預覽所選裝置類型上的體驗外觀。

    切換體驗狀態

    頁內體驗具有不同的視覺狀態，這些狀態會隨著觀看者與體驗的交互而改變。按一下頁面頂端的連結，即可自訂這些狀態的樣式。

    下列狀態可供自訂：

    • 播放之前 -在任何視頻播放開始之前首次渲染時的體驗如何顯示
    • 播放-播放影片時體驗的顯示方式
    • 比賽后 -播放視頻後體驗如何顯示（假設下一個視頻無法自動播放）

    若要設定這些狀態的樣式，請按一下狀態，然後依需要自訂型式。

    選取主題

    主題提供了一組預定義的顏色和字體，可用於快速設置樣式頁內體驗。圖庫提供了一組默認主題，但您也可以創建自己的自定義主題跨帳戶共享。將主題套用至體驗後，您就可以使用樣式設定選項進一步自訂外觀。目前的主題會顯示在樣式選單的頂端。在此示例中，光主題被選中。

    將主題應用於頁內體驗， 按著這些次序：

    1. 點擊左側導航中的當前主題以打開主題選擇器。
    2. 點擊主題以選擇它。預覽將更新以顯示主題顏色。
    3. 完成後，點擊<返回樣式頁面頂部的鏈接。

    點擊重置默認主題鏈接會將默認主題應用於體驗。

    確定要設定樣式的物件

    若要判斷要設定哪個物件樣式，請嘗試將滑鼠暫留在體驗上的物件上。大多數情況下，一個綠色方塊會圍繞物體。該物件的樣式將會變成啟用狀態，且所有不適用的樣式都會變灰。例如，將鼠標懸停在疊加層說明上會同時突出顯示覆蓋和疊加說明樣式。的覆蓋樣式控制疊加層的外觀，包括疊加層標題和疊加層描述。的疊加說明控制僅視頻描述對象的外觀。使用覆疊時，可以設定整個覆疊的樣式，也可以個別設定標題和描述的樣式。

    如果將鼠標懸停在對像上時沒有綠色框，則表明該對像被視為對象的一部分。全球樣式，可以使用全球樣式。可以使用樣式設置已添加到體驗中的文本對象。全球樣式，除非在將文本放置在體驗上時使用文本編輯器更改了任何樣式。

    變更型式

    樣式面板用於自訂體驗中物件的樣式。樣式面板可用於自訂：

    • 文字顏色和背景色
    • 字形
    • 字體大小

    一種自定義CSS文件也可以指定。

    變更文字和背景顏色值

    若要選取色彩值，請依照下列步驟執行：

    1. 按一下色圈以開啟檢色器。
    2. 使用顏色滑桿來設定色彩範圍。
    3. 使用不透明度滑桿設定顏色的不透明度。
    4. 在色彩方塊中按一下以選取顏色。
    5. 按一下 [ X ] 以關閉檢色器。

    您也可以透過輸入顏色代碼來指定顏色：

    • 十六進制 -使用#RRGGBB指定十六進制顏色值。RR（紅色），GG（綠色）和 BB（藍色）是 00 和 FF 之間的十六進制整數，指定顏色的強度。
    • RGBA -RGBA顏色值指定為：rgba（紅色，綠色，藍色，alpha）。Alpha 參數是介於 0.0 (完全透明) 到 1.0 (完全不透明) 之間的數字。
    • HSLA -HSLA顏色值指定為：hsla（色相，飽和度，亮度，alpha）。

    點擊上/下圖標（）在色彩模式之間切換。

    選取字型

    目前的字體將顯示在樣式面板中。使用下拉式清單來選取新字型。點擊X關閉字體選擇器。

    選取字型大小

    目前的字體大小會顯示在樣式面板中。使用下拉式清單來選取新的字型大小。點擊X關閉字體大小選擇器。

    任何已變更的樣式都會顯示復原箭頭。

    點擊撤消箭頭（）將樣式重置為原始值。您還可以通過以下方式重置所有樣式：重設所有默認值在樣式面板的底部。

    新增自訂字型

    若要將自訂字型新增至字型選擇器：

    1. 按一下左側導覽中的任何字型下拉式清單。
    2. 點擊添加自定義字體鏈接。
    3. 輸入您的 CSS 文件的網址。
    4. 確認字型已新增至字型選取器的頂端。
    注意：可以從以下位置獲取開源字體的集合： https://fonts.google.com。
    注意：如果 CSS 檔案位於 CORS 後面，或有其他類型的檔案限制，則該字型將無法正確顯示在編寫體驗、預覽頁面或可共用連結上，因為該 URL 尚未列入白名單以允許使用字型。如果您使用的是自訂字體，請確定該字體可以由體驗存取。
    注意：頁內體驗目前不支援 base64 內嵌字型。解決方法是通過 CSS URL 添加自定義 CSS 文件，改為覆蓋體驗上的字體。

    使用自訂 CSS 檔案

    若要使用自訂 CSS 檔案中的樣式：

    1. 請點擊自定義CSS在左側導航欄中。
    2. 輸入您的 CSS 文件的 URL 或粘貼到 CSS 代碼。
    3. 按一下「儲存」。
