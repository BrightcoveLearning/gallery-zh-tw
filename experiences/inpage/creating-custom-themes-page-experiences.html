為頁內體驗創建自定義主題

{{ page.description }}

主題提供了一組預定義的顏色和字體，可用於快速設置樣式頁內體驗。資源庫提供了一組預設佈景主題，但您也可以建立自己的自訂主題。任何建立的自訂佈景主題都會透過 Video Cloud 帳戶共用。將主題應用到頁內體驗，然後可以使用樣式選項進一步自定義體驗的外觀。

建立自訂佈景主題

若要建立自訂佈景主題，請依照下列步驟執行：

筆記：如果您有現有的主題，可能會更快複製一個主題然後進行編輯與創建新主題。0
  1. 編輯經驗然後點擊風格在左側導航欄中。目前的主題會顯示在樣式選單的頂端。
  2. 按一下主題。將顯示所有主題的列表。
  3. 點擊添加自定義主題關聯。
  4. 進入一個名稱為主題。
  5. 視需要編輯型式。將自動儲存對型式設定所做的所有變更。
  6. 完成後，單擊<返回樣式。

將主題應用於頁內體驗

將主題應用於頁內體驗， 按著這些次序：

  1. 編輯項目，然後單擊風格在左側導航欄中。目前的主題會顯示在樣式選單的頂端。
  2. 按一下主題。將顯示所有主題的列表。點擊一個新的主題來選擇它。
  3. 完成後，單擊<返回樣式。

編輯主題

當您編輯主題時，任何樣式更改都將應用於該主題已應用於的任何現有的頁內體驗除非在體驗中更改了特定的樣式設置。例如，創建一個公司主題，企業主題，並且重疊文字顏色設置為紅色。系統會建立新的「頁面內體驗」，並將覆疊文字顏色變更為藍色。如果有人申請企業主題為了獲得這種新體驗，所有樣式更改都將替換為企業主題除了重疊式文字的顏色，因為該樣式之前已更改。

若要編輯佈景主題，請依照下列步驟執行：

筆記：無法編輯內置主題。
  1. 編輯經驗然後點擊風格在左側導航欄中。目前的主題會顯示在樣式選單的頂端。
  2. 按一下主題。將顯示所有主題的列表。
  3. 單擊編輯圖標 () 對於要編輯的主題。
  4. 視需要編輯型式。將自動儲存對型式設定所做的所有變更。
  5. 完成後，單擊<返回樣式。

複製主題

若要複製佈景主題，請依照下列步驟執行：

  1. 編輯經驗然後點擊風格在左側導航欄中。目前的主題會顯示在樣式選單的頂端。
  2. 按一下主題。將顯示所有主題的列表。
  3. 單擊複製圖標 () 對於要復制的主題。
  4. 輸入主題的新名稱，然後單擊保存。
  5. 視需要編輯型式。將自動儲存對型式設定所做的所有變更。
  6. 完成後，單擊<返回樣式。

刪除主題

若要刪除佈景主題，請依照下列步驟執行：

筆記：無法刪除內置主題。
  1. 編輯經驗然後點擊風格在左側導航欄中。目前的主題會顯示在樣式選單的頂端。
  2. 按一下主題。將顯示所有主題的列表。
  3. 按一下刪除圖示 () 對於要刪除的主題。系統會提示您確認刪除。
  4. 完成後，單擊<返回樣式。
