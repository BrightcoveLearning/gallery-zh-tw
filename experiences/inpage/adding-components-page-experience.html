    將元件新增至頁面內體驗

    在本主題中，您將學習如何將元件新增至頁面內體驗。

    您可以將元件新增至頁面內體驗，讓您自訂並品牌體驗。要將組件添加到頁內體驗中，請編輯體驗，然後單擊“確定”。佈局在左側導航欄中。

    點擊加號圖標（）要添加組件的位置，然後選擇要添加組件的類型。

    可以加入下列類型的元件：

    • 廣告
    • 聊天室
    • 自訂 HTML
    • 圖片
    • 文本
    • 3播放
    • Twitter的飼料
    注意：對體驗進行任何變更之後，請確定您重新發佈體驗，以便變更會出現在顯示體驗的網站上。

    新增廣告元件

    若要在體驗中新增廣告：

    1. 請點擊廣告在組件選擇器上。
    2. 進入一個名稱廣告。
    3. 輸入標頭廣告代碼和廣告單元代碼。標頭廣告代碼將在體驗中的所有廣告之間共享。
    4. 選取應顯示廣告的時間。
    5. 按一下「儲存」。廣告將出現在體驗上。
    注意：如果您有多個廣告的頁內體驗，請確保每個廣告單元代碼有一個獨特的div ID。您可能需要編輯的ID div和的容器參數adblock1匹配該ID。

    新增聊天元件

    您可以在體驗中新增第三方聊天小工具，以提供聊天功能。若要將聊天元件新增至體驗：

    1. 請點擊聊天室在組件選擇器上。
    2. 進入一個名稱用於聊天組件。
    3. 輸入聊天 HTML 代碼。
    4. 選擇顯示聊天的時間。
    5. 按一下「儲存」。聊天將在體驗上呈現。

    新增自訂 HTML 元件

    自訂元件可用來將您自己的自訂 HTML 新增至體驗。例如，如果您已使用行銷自動化平台 (MAP) 建立自訂潛在客戶表單，則可以使用自訂 HTML 元件將來自 MAP 的潛在客戶表單 HTML 新增至頁面內體驗。

    若要將自訂 HTML 新增至體驗：

    1. 按一下元件選擇器上的 [ 自訂 ]。
    2. 輸入 HTML 元件的「名稱」。
    3. 輸入要新增至體驗的自訂 HTML。
    4. 選擇何時應該顯示 HTML。
    5. 按一下「儲存」。HTML 將在體驗上呈現。

    如果沒有輸入 HTML，HTML 佔位符將被顯示。

    注意：基於安全性考量，資源庫編寫環境會從自訂 HTML 元件中刪除指令碼。預覽和發佈頁面內體驗時，指令碼將會執行。這可能會導致自定義 HTML 組件看起來不同，特別是如果它們有很多 JavaScript 內容。

    新增影像元件

    若要將影像新增至體驗：

    1. 請點擊圖片在組件選擇器上。
    2. 進入一個名稱為圖像。
    3. 若要指定影像，您可以：
      • 從檔案系統拖曳影像，然後將其放置到放置區域
      • 按一下瀏覽並從檔案系統中選取影像
      • 輸入遠端影像檔案的 URL
    4. 設定影像大小。
      • 圖像將調整為容器 -比它們所在的容器大的圖像將縮小以適應。較小的影像不會升頻。
      • 寬度和高度-手動輸入影像的寬度和高度
    5. 設定影像對齊方式。
    6. 輸入點選網址。這是點擊圖像時要轉到的 URL。您也可以在新視窗中開啟連結。
    7. 選擇何時應顯示影像。
    8. 按一下 [ 儲存 ] 將影像新增至體驗。

    如果未選取任何影像，將顯示影像預留位置。

    加入文字元件

    若要在體驗中新增文字：

    1. 請點擊文本在組件選擇器上。
    2. 輸入要新增至體驗的文字。工具列可用於格式化文字。
    3. 進入一個名稱為文本。
    4. 選擇何時應顯示文字。
    5. 請點擊保存添加文本。文字會出現在體驗上。

    如果未輸入任何文本，則為文本佔位符將顯示。

    新增 3 播放組件

    3 播放組件可以用來為體驗添加 3Play 互動式記錄。僅當您的視頻字幕是由以下人員生成的時，此組件才有效3播放媒體。當將此組件添加到體驗中時，顯示互動字幕鏈接將會出現。按一下後，影片字幕便會出現。在以下示例中，在視頻播放器下方添加了3Play組件，並將其配置為在玩州。

    有關實時示例，請參見這個連結並播放視頻。

    注意：要為您的帳戶啟用3Play組件，請聯繫您的客戶經理或Brightcove支持。

    若要將 3Play 組件新增至體驗，請執行下列動作：

    1. 請點擊3播放在組件選擇器上。
    2. 進入一個名稱 3Play組件。
    3. 輸入您的3播放項目ID。要找到您的3Play項目ID，請登錄3Play帳戶，然後點擊賬戶信息頁面標題中的鏈接。的專案編號將出現在該頁面上。
    4. 選取應顯示 3Play 組件的時間。
    5. 請點擊保存添加3Play組件。3Play 組件將出現在體驗中。

    新增推特動態消息元件

    若要將 Twitter 動態消息新增至體驗：

    1. 請點擊Twitter的飼料在組件選擇器上。
    2. 進入一個名稱用於Twitter Feed。
    3. 在推特網站上創建一個小部件。如需建立 Twitter 元件的說明，請參閱 Twitter 開發人員文件。
    4. 複製 Twitter 小工具的 HTML，然後將其貼到 Twitter 動態消息設定中。
    5. 選取何時應顯示 Twitter 動態消息。
    6. 請點擊保存添加Twitter feed。Twitter 動態消息將出現在體驗中。
