向頁內體驗添加組件

{{ page.description }}

您可以將元件新增至頁面內體驗，讓您自訂並品牌體驗。要將組件添加到頁內體驗中，請編輯體驗，然後單擊“確定"。佈局在左側導航欄中。

單擊加號圖標 () 要添加組件的位置，然後選擇要添加的組件類型。

可以加入下列類型的元件：

  • 廣告
  • 聊天
  • 自定義HTML
  • 圖像
  • 文本
  • 3播放
  • Twitter的飼料
筆記：對體驗進行任何更改後，請確保您重新發布經驗因此更改會出現在顯示體驗的網站上。

新增廣告元件

若要在體驗中新增廣告：

  1. 點擊廣告在組件選擇器上。
  2. 進入一個名稱廣告。
  3. 輸入標頭廣告代碼和廣告單元代碼。標頭廣告代碼將在體驗中的所有廣告之間共享。
  4. 選取應顯示廣告的時間。
  5. 點擊保存。廣告將出現在體驗上。
筆記：如果您有多個廣告的頁內體驗，請確保每個廣告單元代碼有一個獨特的div ID。您可能需要編輯div和容器參數adblock1匹配那個id。

新增聊天元件

您可以在體驗中新增第三方聊天小工具，以提供聊天功能。若要將聊天元件新增至體驗：

  1. 點擊聊天在組件選擇器上。
  2. 進入一個名稱用於聊天組件。
  3. 輸入聊天 HTML 代碼。
  4. 選擇顯示聊天的時間。
  5. 點擊保存。聊天將在體驗上呈現。

新增自訂 HTML 元件

自訂元件可用來將您自己的自訂 HTML 新增至體驗。例如，如果您已使用行銷自動化平台 (MAP) 建立自訂潛在客戶表單，則可以使用自訂 HTML 元件將來自 MAP 的潛在客戶表單 HTML 新增至頁面內體驗。

若要將自訂 HTML 新增至體驗：

  1. 點擊風俗在組件選擇器上。
  2. 進入一個名稱用於HTML組件。
  3. 輸入要新增至體驗的自訂 HTML。
  4. 選擇何時應該顯示 HTML。
  5. 點擊保存。HTML 將在體驗上呈現。

如果沒有輸入 HTML，HTML 佔位符將被顯示。

筆記：基於安全性考量，資源庫編寫環境會從自訂 HTML 元件中刪除指令碼。預覽和發佈頁面內體驗時，指令碼將會執行。這可能會導致自定義 HTML 組件看起來不同，尤其是當它們包含大量 JavaScript 內容時。

新增影像元件

若要將影像新增至體驗：

  1. 點擊圖像在組件選擇器上。
  2. 進入一個名稱為圖像。
  3. 若要指定影像，您可以：
    • 從檔案系統拖曳影像，然後將其放置到放置區域
    • 點擊瀏覽然後從文件系統中選擇一個圖像
    • 輸入遠端影像檔案的 URL
  4. 設置圖像尺寸。
    • 圖片將調整為容器大小 -尺寸大於其所在容器的圖像將按比例縮小以適合。較小的影像不會升頻。
    • 寬度和高度 -手動輸入圖像的寬度和高度
  5. 設置圖像結盟。
  6. 進入一個點擊後到達網址。這是點擊圖像時要轉到的 URL。你也可以在新窗口中打開鏈接。
  7. 選擇何時應顯示影像。
  8. 點擊保存將圖像添加到體驗中。

如果未選取任何影像，將顯示影像預留位置。

加入文字元件

若要在體驗中新增文字：

  1. 點擊文本在組件選擇器上。
  2. 輸入要新增至體驗的文字。工具列可用於格式化文字。
  3. 進入一個名稱為文本。
  4. 選擇何時應顯示文字。
  5. 點擊保存添加文本。文字會出現在體驗上。

如果未輸入文本，則為文本佔位符 將顯示。

新增 3 播放組件

3 播放組件可以用來為體驗添加 3Play 互動式記錄。僅當您的視頻字幕是由以下人員生成的時，此組件才有效3播放媒體。當將此組件添加到體驗中時，顯示互動式筆錄鏈接將會出現。按一下後，影片字幕便會出現。在以下示例中，在視頻播放器下方添加了3Play組件，並將其配置為在玩耍狀態。

有關實時示例，請參見這個連結並播放視頻。

筆記：要為您的帳戶啟用 3Play 組件，請聯繫您的客戶經理或Brightcove 支持.

若要將 3Play 組件新增至體驗，請執行下列動作：

  1. 點擊3播放在組件選擇器上。
  2. 進入一個名稱 3Play組件。
  3. 輸入您的3播放項目ID。要找到您的3Play項目ID，請登錄3Play帳戶，然後點擊賬戶信息頁面標題中的鏈接。這專案編號將出現在該頁面上。
  4. 選取應顯示 3Play 組件的時間。
  5. 點擊保存添加3Play組件。3Play 組件將出現在體驗中。

新增推特動態消息元件

若要將 Twitter 動態消息新增至體驗：

  1. 點擊Twitter的飼料在組件選擇器上。
  2. 進入一個名稱用於Twitter Feed。
  3. 在Twitter站點上創建一個小部件。有關創建Twitter小部件的幫助，請參閱Twitter開發人員文檔。
  4. 複製 Twitter 小工具的 HTML，然後將其貼到 Twitter 動態消息 設定中。
  5. 選取何時應顯示 Twitter 動態消息。
  6. 點擊保存添加Twitter feed。Twitter 動態消息將出現在體驗中。
