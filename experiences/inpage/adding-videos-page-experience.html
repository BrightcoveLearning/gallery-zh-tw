將視頻添加到頁內體驗

{{ page.description }}

您可以將視訊雲端影片和 YouTube 影片新增至頁面內體驗。要將視頻添加到頁內體驗中，請編輯體驗，然後單擊影片在左側導航欄中。

筆記：在將 YouTube 視頻添加到體驗之前，您必須配置 YouTube 設置.

點擊新增影片打開視頻選擇器，然後選擇一個來源：

  • 視頻雲 -將使用視頻雲視頻內容
  • YouTube -將使用YouTube視頻內容（請參閱YouTube整合詳情有關詳細信息，請參見本文檔的“部分"）
筆記：可以將來自 Video Cloud 和 YouTube 的視頻添加到同一個頁內體驗。

如果YouTube來源被選中後，將會出現一個頻道選擇器，您可以選擇要使用的YouTube頻道。

當視頻雲來源被選中，選擇一種應如何添加視頻的方法：

  • 選擇視頻 -視頻將被手動選擇和使用
  • 選擇播放列表 -將使用播放列表中的視頻
  • 使用參考編號 -將使用來自播放列表參考ID的視頻

視頻被選中後，它們將顯示在頁面的正文中。在此範例中，使用視訊雲端播放清單。

筆記：對體驗進行任何更改後，請確保您重新發布經驗因此更改會出現在顯示體驗的網站上。

手動選取視訊

要手動選擇視頻，請選擇一個方法的選擇視頻。使用視訊雲端來源，視訊選擇器會顯示您的 Video Cloud 帳戶中 100 個最近更新的影片清單。當您向下翻頁時，將會擷取新影片。使用 YouTube 來源，所選頻道中的所有視頻都將被顯示。按一下影片將其選取，再按一下即可取消選取。使用搜索字段搜索視頻。在此範例中，已選取視訊雲端來源。

完成後，單擊添加。

搜尋影片

這搜索字段可用於搜索視頻。搜尋 Video Cloud 影片時，圖庫會搜尋影片名稱、簡短和完整說明、標籤和自訂欄位。搜尋 YouTube 影片時，「圖庫」會使用 YouTube 搜尋應用程式介面。

點擊更多搜尋選項以便更好地控制搜索哪些字段以及包含和排除文本。請注意，只有在使用視訊雲端來源時，才能使用此選項。

使用播放清單

要使用播放列表中的視頻，請選擇一個方法的選擇播放列表。透過視訊雲端來源，影片選擇器會顯示您的 Video Cloud 帳戶中 20 個最近更新的播放清單清單。使用 YouTube 來源，視頻選擇器將顯示所選頻道中所有播放列表的列表。搜尋欄位可用來搜尋播放清單。在此範例中，已選取視訊雲端來源。

請注意，只能選擇一個播放清單。完成後，單擊選擇。

使用播放清單參考 ID

要通過指定播放列表參考ID來使用播放列表中的視頻，請選擇一個方法的使用參考編號。只有在使用視訊雲端來源時才支援此選項。視頻選擇器提供了一個字段，可輸入播放列表參考ID。透過使用播放清單參考 ID，使用媒體模組將參考 ID 重新指定給其他播放清單，即可輕鬆變更體驗中的影片。在以下情況下，無需編輯和重新發布體驗 您需要更改體驗中顯示的視頻。

請注意，只能使用單一播放清單參考 ID。完成後，單擊選擇。

將互動功能新增至視訊

互動和伴隨組件可以新增至視訊，並在播放視訊時顯示。互動包括鏈接和牌並在播放視頻時作為播放器的一部分顯示。隨播組件包括的HTML，圖片和文本對象，並顯示在播放器外部。

已配置互動功能的視頻將在互動性指示互動類型的列。在以下示例中，關聯和卡片互動和隨播圖片已配置。

如需新增和設定互動與隨附元件的相關資訊，請參閱：

  • 將互動添加到頁內體驗
  • 配置伴隨組件以實現頁內體驗

從頁面內體驗移除視訊

從頁內體驗中刪除視頻的過程取決於方法選擇添加視頻：

  • 選擇視頻 -選中要刪除的視頻旁邊的複選框，然後單擊刪除視頻。您也可以單擊刪除圖標 (） 在裡面行動柱子。
  • 選擇播放列表 - 點擊刪除播放列表刪除所有視頻
  • 使用參考編號 - 點擊刪除播放列表刪除所有視頻

刪除視頻後，該視頻上配置的所有互動或伴隨組件將不會被刪除。如果視訊再重新加入體驗中，互動和伴隨組件將會重新出現。
