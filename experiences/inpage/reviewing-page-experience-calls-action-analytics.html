查看頁面內體驗號召性用語分析

{{ page.description }}

行動呼叫分析為頁面內體驗提供點擊次數和表單轉換的分析。要查看號召性用語分析，請編輯體驗並單擊體驗分析>號召性用語在左側導航欄中。

選取時間範圍

若要指定報告的時間週期，您可以按一下快速日期或使用行事曆控制項。

使用快速日期

頁面頂端有一系列快速日期。可用的快速日期如下：

  • 今天
  • 昨天
  • 最近7天
  • 最近30天
  • 這個月 -迄今為止的日曆月-例如，如果今天是3月17日，則此報告將顯示3月1日-3月17日
  • 上個月 -前一個日曆月-例如，如果今天是2月14日，則此報告將顯示1月1日-1月31日。

按一下快速日期以更新報告。

使用行事曆控制項

您也可以按一下行事曆控制項來選取開始和結束日期。

點擊申請更新報告。

檢閱點閱分析

點選分析會追蹤頁面內體驗中的互動性。下列物件將觸發要追蹤的點擊次數分析：

  • 使用連結 URL 設定的連結互動
  • 使用連結 URL 設定的卡片互動
  • 使用可點選 URL 設定的映像元件

要查看點擊率分析，請單擊體驗分析>號召性用語在左側導航中，然後單擊點擊率在頁面標題中。

點選分析會顯示下列體驗資料：

  • 號召性用語名稱 -號召性用語的名稱：
    • 連結-顯示連結標題
    • 卡-顯示卡片標題
    • 影像-顯示建立期間指定的名稱
    • 自訂 HTML-顯示建立期間指定的名稱
  • 更新 -互動的最新更新日期和時間
  • 類型 -互動類型，鏈接或卡片
  • 印象 - 互動被顯示的次數（有卡片，當卡片圖標（) 出現；無需展示卡片即可記錄印象）
  • 點擊率 -單擊鏈接的次數，單擊卡中的鏈接或單擊圖像組件上的點擊後到達URL的次數
  • 點擊率 （％） -點擊鏈接的觀看者所佔的百分比（點擊次數除以展示次數）

檢閱表單轉換分析

表單轉換分析會在頁面內體驗中追蹤表單提交和自訂 HTML 連結活動。下列物件會觸發要追蹤的表單轉換分析：

  • 從 HTML 附屬元件提交的表單
  • 從自訂 HTML 元件提交的表單
  • 自訂 HTML 元件中的可點選 URL

要查看表單轉換分析，請單擊體驗分析>號召性用語在左側導航中，然後單擊表格轉換在頁面標題中。

表單轉換分析會顯示下列體驗資料：

  • 號召性用語名稱 -組件的名稱（在將組件添加到體驗中時分配名稱）
  • 更新 -組件的最後更新日期和時間
  • 類型 - 交互類型（HTML 伴隨組件用 (） 圖標
  • 印像數 -顯示組件的次數
  • 表格轉換 -提交表單或單擊自定義HTML組件中的URL的次數
  • 表格轉換率（％） -提交表單的觀看者所佔的百分比（表單轉化除以展示次數）
