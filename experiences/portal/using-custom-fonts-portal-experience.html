在門戶體驗中使用自定義字體

{{ page.description }}

圖庫網站編輯器提供了以下功能：定制風格通過更改主題，顏色和字體來實現門戶體驗。也可以在Portal Experience中使用您自己的自定義字體。在本主題中， Acme的字體Google字體目錄將用於定制門戶體驗。

鏈接到自定義字體文件

這Google字體目錄提供可在網頁上使用的開源字體。找到要使用的字體，然後單擊選擇此款式。在本主題中， Acme字體將被使用。

根據Google的幫助，應在文檔中嵌入以下幾行。

<link rel="preconnect" href="https://fonts.gstatic.com"> 
<link href="https://fonts.googleapis.com/css2?family=Acme&display=swap" rel="stylesheet">

要將這些行包含在“門戶體驗"中，請按照下列步驟操作：

  1. 編輯體驗。
  2. 點擊外觀和行為>頁眉和頁腳在左側導航欄中。
  3. 粘貼<link>標記到自定義標題HTML場地。
  4. 點擊保存。

為自定義字體創建CSS

要使用自定義字體，style標籤將被添加到門戶自定義標題HTML字段以設置適當的文本對象的樣式。

瀏覽器隨附的開發工具有助於獲取頁面上要設置樣式的對象（我們要為其更改字體的對象）的類名。請注意，元素名稱將根據所使用的模板進行更改。在下面的示例中，我們將更改精選視頻的字體和頁面上視頻網格中顯示的視頻名稱所使用的字體。屏幕截圖是使用Google Chrome瀏覽器拍攝的。

  1. 使用瀏覽器開發工具，檢查您要更改字體的體驗元素。在這種情況下，正在檢查精選視頻標題。找到對象的類名稱。
  2. 將適當的CSS添加到自定義標題HTML字段以使用新字體設置特色視頻標題的樣式。 
      
      <style>
        .video-name {
         font-family:Acme;
       }
      </style>
  3. 在網格中找到視頻標題的類別名稱。
  4. 在現有的內部添加適當的CSS <style>標籤以使用新字體在網格中為視頻標題設置樣式。 
    
      .bc-video-browser .bc-video-grid-items .bc-video-grid-item .caption {
       font-family:Acme;
    }
  5. 確認自定義標題HTML字段包含以下內容。 
    <link rel="preconnect" href="https://fonts.gstatic.com"> 
    <link href="https://fonts.googleapis.com/css2?family=Acme&display=swap" rel="stylesheet">
  <style>
      .video-name {
      font-family:Acme;
    }

    .bc-video-browser .bc-video-grid-items .bc-video-grid-item .caption {
    font-family:Acme;
  }
</style>
    筆記：也可以將 CSS 保存在單獨的文件中，並使用自定義 CSS 網址場地。
  6. 點擊保存。
  7. 點擊預覽>響應式預覽驗證是否顯示了更新的字體。
  8. 完成後，發布經歷。

模板的完整CSS文件可通過單擊查看CSS作為鏈接的一部分外觀和行為>自定義設置。如果要更改字體的所有實例，則可以搜索所有出現的font-family在模板CSS中，然後將適當的條目添加到您的<style>標籤。
