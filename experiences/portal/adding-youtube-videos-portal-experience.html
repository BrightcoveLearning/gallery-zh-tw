將YouTube視頻添加到門戶體驗

{{ page.description }}

使用圖庫集合編輯器，您可以選取要在入口網站體驗中提供哪些視訊。透過入口網站體驗，影片會整理成收藏。收藏集可以包含來自視頻雲帳戶或來自YouTube頻道。本主題將著重於將 YouTube 影片新增至收藏。有關將Video Cloud視頻添加到集合中的信息，請參閱將視頻雲視頻添加到門戶體驗。

YouTube 整合要求

  • 在將YouTube視頻添加到視頻網站之前，您必須配置YouTube設置
  • 只有 YouTube 播放清單中的影片才能包含在收藏中。不屬於 YouTube 播放清單的影片將無法透過「圖片瀏覽」選集管理員存取。播放清單也必須將播放清單隱私權設定設定為「公開」。
  • 私人影片不應放在公開播放清單中

YouTube 整合限制

  • 無法下載 YouTube 影片
  • 使用 IMA 外掛程式在布萊特灣播放器播放的 YouTube 影片不會在行動裝置上顯示廣告
  • 搜尋時，所有收藏都必須包含 YouTube 影片，而影片必須由所選頻道擁有
  • ' 使用 YouTube 的客戶須遵守 YouTube 的條款及細則約束

建立集合

視訊網站中的視訊會組織成集合。若要建立集合，請依照下列步驟執行：

  1. 點擊影片在左側導航欄中。管理收藏頁面隨即開啟。頁面右側會顯示所選範本類型的首頁視訊配置預覽。
  2. 輸入集合的名稱，然後按進入。此名稱會顯示給檢視者，因此請務必使用有意義的名稱。單擊編輯 () 圖標來編輯集合名稱。

編輯集合

  1. 若要編輯集合，請按一下集合名稱。
  2. 編輯收藏頁面隨即開啟。[編輯集合] 頁面可用來新增視訊至集合，以及設定其他集合屬性。這變更集合下拉列表將列出所有已創建的集合。選取要編輯的集合。右側的預覽會顯示系列在「首頁」上顯示的位置。
  3. 這館藏來源指定該集合的視頻內容的來源：
    • 視頻雲 -將使用視頻雲視頻內容。有關信息，請參閱將視頻雲視頻添加到門戶體驗。
    • YouTube -將使用YouTube視頻內容。
  4. 選擇YouTube 渠道使用。
  5. 選擇一個播放清單。只有播放清單中的影片才會顯示。
  6. 點擊保存。

新增收集資訊

視選取的範本而定，可以設定收集詳細資訊。要添加集合詳細信息，請點擊收集信息部分。收集資訊區段提供新增集合影像和集合描述的功能。一些社交網絡將使用圖像和描述，當體驗的鏈接發布到社交媒體網站時。

這收藏圖片用於表示圖庫網站上的收藏。如果未指定集合影像，「圖庫」會使用集合中第一個視訊的視訊縮圖來指定一個集合影像。要添加集合圖像，請單擊瀏覽並從文件系統中選擇一個圖像。您也可以將影像拖放到指定的區域。

這集合描述用於描述集合。

常見問題

1.如何回報分析？

數據分析模組會將所有 YouTube 影片的播放次數分組，並將這些影片報告為一個項目。

2.系列可以同時包含 YouTube 和影片雲端來源的影片嗎？

沒有每個「圖庫」集合只能包含來自一個來源的影片。不過，「圖片瀏覽」網站可以包含多個收藏，因此您可以擁有包含 YouTube 和影片雲端的內容的「圖片瀏覽」網站。

3.圖片庫中的 YouTube 影片必須來自我自己的頻道，還是可以在其他出版商的頻道中使用影片？

您可以使用您自己頻道中的影片，以及您連結至任何「精選頻道」的影片。如此一來，擁有多個頻道的公司 (全部連結在 YouTube 上)，就能在單一「圖片庫」網站中加入任意數量的影片。您也可以在「圖片瀏覽」網站中加入不屬於您的頻道的影片。

4.如何在圖片庫中使用 YouTube 影片中繼資料？

圖片庫會使用下列 YouTube 中繼資料：

  • 識別碼
  • 標題
  • 描述
  • 高品質的縮圖網址
  • 持續時間
  • 發佈日期

此元數據的使用方式與Video Cloud視頻中的元數據相同，將其添加到<meta>頭部中的標籤，站點地圖中使用的標籤等。

5.如果為圖片庫網站設定的布萊特灣播放器是具有觀眾功能的播放器，觀眾模組會擷取 YouTube 影片的觀看資料嗎？

是的也支援潛在客戶表單。

6.YouTube 影片是否支援「圖片庫」潛在客戶表單？

是的

7.在「圖片瀏覽」網站中可以使用哪些 YouTube 影片是否有任何限制？例如，如果 YouTube 視頻不「允許嵌入」，可以使用該視頻嗎？另外，如果影片被標記為非公開或私人，該怎麼辦？

是的
