    將視訊雲端視訊新增至入口網站體驗

    在本主題中，您將學習如何將視訊雲端視訊新增至入口網站體驗。

    使用圖庫集合編輯器，您可以選取要在入口網站體驗上顯示的視訊。透過入口網站體驗，影片會整理成收藏。收藏集可以包含來自Video Cloud帳戶或來自YouTube頻道。本主題將著重於將 Video Cloud 視訊新增至收藏。有關將YouTube視頻添加到收藏集的信息，請參見將YouTube視頻添加到門戶體驗。

    建立集合

    入口網站體驗中的影片會整理成收藏。若要建立集合，請依照下列步驟執行：

    1. 按一下左側導覽中的 [ 影片 ]。管理收藏頁面隨即開啟。頁面右側會顯示所選範本類型的首頁視訊配置預覽。
    2. 輸入集合的名稱，然後按 Enter 鍵。此名稱會顯示給檢視者，因此請務必使用有意義的名稱。按一下「編輯」( ) 圖示以編輯集合名稱。

    編輯集合

    1. 若要編輯集合，請按一下集合名稱。
    2. 編輯收藏頁面隨即開啟。[編輯集合] 頁面可用來新增視訊至集合，以及設定其他集合屬性。「變更集合」下拉式清單會列出所有已建立的集合。選取要編輯的集合。右側的預覽會顯示系列在「首頁」上顯示的位置。
    3. 集合來源指定的視頻內容的集合的來源：
      • 視頻雲 -視頻雲視頻內容將被使用
      • 視頻內容將被使用。有關信息，請參見將YouTube視頻添加到門戶體驗。
    4. 使用Video Cloud Collection Source，視頻可以來自播放清單或使用自訂分配：
      • 播放清單 -集合中的視頻基於“視頻雲”播放列表
      • 自訂 -系列中的影片以影片標籤、自訂搜尋或自訂中繼資料欄位為基礎
    注意：輸入的值包含標籤和排除標籤作為站點功能>站點搜索設置也將應用於創建的任何集合。這可能會限制您在建立收藏時顯示的視訊。
    注意：如果您最近變更視訊中繼資料或播放清單，這些變更最多可能需要 6 分鐘才會出現在「集合編輯器」中，以及已發佈的體驗上。

    使用播放清單影片指派

    當您選擇使用播放清單視頻分配，集合中的視頻將基於視頻雲播放列表。

    1. 選取播放清單選項。您的影片雲端帳戶中的播放清單將會出現。
    2. 使用導航控件（）或搜索功能以找到所需的播放列表。選取播放清單後，將會顯示該播放清單中的影片清單。
      注意：收藏中的視訊順序取決於播放清單設定。
    3. (選擇性) 視選取的範本而定，可設定其他收集詳細資訊。如需詳細資訊，請參閱本主題的〈新增集合資訊〉一節。
    4. 按一下 [ 儲存] 儲存變更。

    使用自訂視訊指派

    當您選擇使用「自訂視訊」工作時，系列中的視訊將會根據視訊標籤、自訂搜尋或自訂中繼資料欄位而定。

    1. 選取「自訂」選項。
    2. 搜尋參數可以設定為「根據標籤」、「搜尋」或「自訂欄位」來包含或排除視訊。
      注意：使用 [自訂欄位] 選項時，您應該輸入自訂欄位的 [ 內部 ] 名稱，而不是 [ 顯示] 名稱。
    3. 設定適當的搜尋參數以傳回所需的視訊。在以下示例中，標有的視頻視頻雲要點將被退回。
    4. （可選）點擊添加圖標（）在適當的部分添加或排除其他參數。
    5. 設定視訊排序順序：
      • 最近一次-視頻是根據啟動時間排序的，最近的視頻列在第一
      • 最舊的優先 -視頻是根據啟動時間排序的，最舊的視頻列在第一
      • 最多檢視次數-最常見的檢視
      • A-Z -按升序排序
      • Z-A -以遞減順序排序
    6. (選擇性) 視選取的範本而定，可設定其他收集詳細資訊。如需詳細資訊，請參閱本主題的〈新增集合資訊〉一節。
    7. 按一下 [ 儲存] 儲存變更。

    新增收集資訊

    視選取的範本而定，可以設定收集詳細資訊。若要新增收集詳細資訊，請按一下收集資訊區段。收集信息部分提供了添加收藏圖片和集合描述。一些社交網絡將使用圖像和描述，當體驗的鏈接發布到社交媒體網站時。

    「集合影像」用來代表「圖庫」網站上的集合。如果未指定集合影像，「圖庫」會使用集合中第一個視訊的視訊縮圖來指定一個集合影像。要添加集合圖像，請單擊瀏覽並從文件系統中選擇一個圖像。您也可以將影像拖放到指定的區域。

    集合描述用於描述集合。
