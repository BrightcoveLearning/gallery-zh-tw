    建立直播活動頁面體驗

    在本主題中，您將學習如何自訂即時活動頁面體驗的內容。

    圖庫提供即時活動頁面內模板，可用於提供現場活動作為頁面體驗的一部分。Brightcove 提供多種提供即時串流視訊的選項。看到傳送直播有關所有選項的列表。如需使用即時模組廣播即時事件的詳細步驟集，請參閱：

    • 逐步：使用即時模組與電流線上傳送
    • 逐步：使用開放式廣播軟體 (OBS) 的直播模組
    注意：即使播放程式設定為不自動播放，即使播放程式的「即時活動頁面體驗」也會在桌面瀏覽器中自動播放。這是經過設計的在倒數計時完成後，模擬工坊會強制玩家進行遊戲，讓坐在頁面上等待活動開始的使用者可以開始觀看，而無需再次點擊播放。現場活動頁面體驗不會在 iOS 或 Android 裝置上自動播放。

    要創建新的現場活動頁內體驗，請創建新的頁內體驗，然後選擇直播活動頁內模板。

    即時事件頁面內範本有一些設定和內容項目不存在於其他頁面內體驗範本。

    1. 現在顯示給查看者
    2. 直播活動狀態
    3. 社交分享
    4. 圖片佔位符
    5. 活動倒計時
    6. 文字佔位符
    7. 日曆提醒
    8. 視頻佔位符

    現在向觀眾顯示

    的現在顯示給查看者設置用於配置哪個現場活動狀態向觀看者顯示。

    請點擊更改並選擇應顯示的狀態。變更此設定之後，不需要重新發佈體驗，但是，狀態變更可能會出現在生產網站上的一分鐘之前。

    注意：您 (發行者) 負責變更即時事件狀態。資源庫永遠不會自動切換即時事件狀態。

    即時事件狀態

    即時活動頁面體驗的體驗狀態與其他頁面內體驗類型不同。按一下頁面頂端的連結，即可自訂上述各狀態的外觀。

    即時活動頁面體驗有下列狀態：

    • 活動前 -直播活動開始前直播體驗將如何出現
    • 生活 -直播活動進行時體驗將如何出現
    • 事件後 -直播活動結束後體驗如何顯示

    要編輯不同的體驗狀態，請單擊狀態，然後為體驗添加組件如所須。的佈局菜單為每個狀態提供了更多的自定義選項。

    社交分享

    社交分享選項提供連結，可分享活動和影片到您的社交媒體帳戶。

    要配置社交共享設置，請點擊配置共享鏈接或點擊設置>社會在左側導航欄中。社交設定包括：

    • 活動分享
    • 影片分享

    活動分享

    活動分享設置允許觀眾將活動頁面分享到社交媒體帳戶。請依照下列步驟設定事件詳細資料：

    1. 要添加一個活動圖片，點擊添加圖片接著：
      • 從檔案系統拖曳影像，然後將其放置到放置區域
      • 按一下瀏覽並從檔案系統中選取影像
      • 輸入遠端影像檔案的 URL
    2. 輸入一個活動標題和活動說明。
    3. 選擇事件共享選項的顯示方式，分享文字或顯示社交圖標。
    4. 選取可用的共用選項。
    5. 輸入頁面網址體驗將嵌入其中的地方。
    6. 按一下「產生」。這將生成可以添加到<head>頁面的一部分。本功能將允許社交網站在觀眾分享您的體驗時顯示事件影像、標題與描述。

    視訊分享

    視頻分享設置允許觀眾將視頻分享到社交媒體帳戶。請依照下列步驟設定視訊共用：

    1. 選擇視頻共享選項的顯示方式，分享文字或顯示社交圖標。
    2. 選取可用的共用選項。
    3. 按一下 [ 儲存] 儲存變更。

    影像預留位置

    活動前和事件後狀態提供顯示英雄影像的能力。

    若要新增主體影像：

    1. 點擊添加圖片鏈接。
    2. 進入一個名稱用於圖像（僅允許使用ASCII字符）。
    3. 若要指定影像，您可以：
      • 從檔案系統拖曳影像，然後將其放置到放置區域
      • 按一下瀏覽並從檔案系統中選取影像
      • 輸入遠端影像檔案的 URL
    4. 設定影像大小。
      • 裁剪圖片 -在中央裁剪圖像，使其占據整個16：9的空間，但是頂部/底部或左側/右側的一些部分會丟失，具體取決於圖像的縱橫比
      • 將圖像包含在組件範圍內 -將圖片包含在16：9的空間範圍內，但根據寬高比，在圖片的上方/下方或左側/右側會出現一些空白
      • 拉伸圖像 -忽略圖像的縱橫比，並強制其完全填充16：9的空間；這會使圖像變形
      • 原始影像尺寸 -使用原始圖像尺寸，不調整尺寸；沒有反應
    5. 選取應顯示影像的狀態。
    6. 按一下 [ 儲存 ] 將影像新增至體驗。

    事件倒數

    倒數計時器會顯示即時活動開始前的月數、天數、小時、分鐘和秒數。倒數計時器範例如下所示。

    注意：倒計時計時器的樣式可以通過以下方式設置樣式：風格在左側導航欄中。

    若要新增倒數計時器，請依照下列步驟執行：

    1. 點擊配置事件倒數頁面上的鏈接。事件設定頁面隨即開啟。
    2. 檢查一下顯示活動倒數。
    3. 設置開始日期，開始時間和活動時區。
    4. 進入一個過渡訊息。當倒數計時器達到零，但直播開始前，將顯示此訊息。
    5. 按一下「儲存」。

    文字預留位置

    事件前和 POST 事件狀態提供顯示靜態文字的功能。

    若要新增靜態文字：

    1. 點擊添加文字鏈接。
    2. 輸入要新增至體驗的文字。工具列可用於格式化文字。
    3. 請點擊保存將文本添加到體驗中。

    3. 設置結束日期，時間結束和活動標題。的活動說明和活動地點是可選的。

    視訊預留位置

    在活動前、現場和活動後狀態中可以顯示不同的影片。視頻雲端視頻和 YouTube 視頻可以添加到體驗中。

    注意：您必須先設定 YouTube 設定，才能將 YouTube 影片加入體驗中。

    要選擇要顯示的視頻：

    1. 點擊選擇影片鏈接或點擊影片在左側導航欄中。的選擇一個視頻對話框將會出現。
    2. 選取要選取視訊的狀態：
      • 事件前
      • 即時
      • 事件後
    3. 選擇視頻資源：
      • 視頻雲 -視頻雲視頻內容將被使用
      • 的YouTube -將使用YouTube視頻內容
    4. 搜索視頻，單擊將其選中，然後單擊選擇。影片將會新增至視訊清單。
    5. 點擊生活要么賽后部分，或單擊選擇視頻選擇其他視頻。

    互動和伴隨組件可以添加到活動前和活動後的視頻，並在播放視頻時顯示。互動包括連結和卡片，並在播放視訊時顯示為播放程式的一部分。隨附的元件包括 HTML 、影像和文字物件，並顯示在播放器之外。

    注意：除非是 VOD 資產，否則即時視訊無法使用互動和隨附元件。

    使用互動設定的視訊會在「互動」欄中顯示一個圖示，指出互動類型。如需新增和設定互動與隨附元件的相關資訊，請參閱：

    • 將互動添加到頁內體驗
    • 配置伴隨組件以實現頁內體驗

    選取即時影片

    當。。。的時候生活狀態（已選中）時，必須輸入實時視頻資產的視頻雲視頻ID或參考ID。實時視頻的來源將始終是視頻雲。

    選擇參考編號選項可讓您更改媒體模塊中實況視頻的參考ID，而不必更改體驗中的視頻ID。例如，您可以選擇使用參考ID表示當前的實時視頻現場活動。創建現場體驗時，選擇參考編號然後輸入現場活動。

    現在，每當有新的直播事件時，請使用媒體模塊將該視頻的參考ID設置為現場活動。

    使用參考 ID，即時體驗不需要在新事件開始時編輯和重新發佈。啟動新事件時，請使用「媒體」模組編輯即時視訊資產的視訊參考 ID。

    搜尋影片

    搜尋欄位可用來搜尋影片。搜尋 Video Cloud 影片時，圖庫會搜尋影片名稱、簡短和完整說明、標籤和自訂欄位。搜尋 YouTube 影片時，「圖庫」會使用 YouTube 搜尋應用程式介面。

    按一下「更多搜尋選項」，可更精確地控制要搜尋的欄位，以及包含和排除文字。請注意，只有在使用視訊雲端來源時，才能使用此選項。
