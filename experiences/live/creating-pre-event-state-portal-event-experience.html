為門戶事件體驗創建事件前狀態

{{ page.description }}

實時事件門戶模板提供事件前，居住，事後，視頻存檔和視頻詳細信息頁面模板。活動前，現場和埠-事件頁面模板可以自定義，使網站在現場活動之前，期間和之後有不同的外觀。事件前狀態是網站在即時事件開始前顯示的內容。

要為網站創建事件前主頁狀態，請創建新的門戶網站體驗 使用實時事件門戶模板。在圖庫網站編輯器中，單擊頁數在左側導航欄中。確認賽前狀態已選擇。

事件前的首頁狀態提供下列內容區域。

  • A-橫幅 -用於添加號召性用語，包括自定義HTML，圖像和廣告。看在門戶網站體驗上配置號召性用語想要查詢更多的信息。
  • B-主頁徽標和名稱 -用於將查看器返回首頁。這裡有兩個佔位符，所以你可以選擇有一個圖像後跟文本例如。圖像將縮小到 55 像素的高度和 495 像素的最大寬度。
  • C-鏈接到視頻存檔 -用於查看已為該網站創建的所有視頻集。請注意，列表中的第一個收藏將被視為「精選收藏」，並會顯示在活動前頁面（下方 H 區）。
  • D-Twitter設置 -提供使用Twitter feed小部件的功能。看配置Twitter提要。
  • E-背景圖片，文字和鏈接 -用於添加背景圖像，字幕文本和鏈接
  • F-倒數計時設定 -提供倒數計時器，直到活動開始為止。看配置倒數計時器。
  • G-活動前消息 -靜態短信
  • H -新增影片 -用於向網站添加視頻收藏。看添加視頻想要查詢更多的信息。
  • 一世 -視頻網格盒 -用於添加號召性用語，包括自定義HTML或圖像。看在門戶網站體驗上配置號召性用語想要查詢更多的信息。
  • J-頁腳對象 -用於向頁面頁腳添加文本和鏈接

有關將圖像，鏈接和靜態文本添加到門戶網站頁面模板的信息，請參閱定制門戶網站體驗頁面的內容。

筆記：網站顏色將根據所選主題而有所不同。

設定推特動態消息

該網站可以配置在頁面上的 Twitter 嵌入。對於可用的嵌入類型，請檢查Twitter網站。一個樣本喜歡的時間軸如下所示。

若要新增 Twitter 動態消息，請依照下列步驟執行：

  1. 在Twitter站點上創建一個小部件。有關創建Twitter小部件的幫助，請參閱Twitter開發人員文檔。
  2. 點擊Twitter設置頁面上的鏈接。
  3. 複製Twitter小部件的HTML並將其粘貼到Twitter的飼料設置。
  4. 點擊保存。

設定倒數計時器

網站可以設定倒數計時器，顯示即時活動開始前的月數、天數、小時和分鐘數。倒數計時器範例如下所示。

倒數到零時，文字即將推出將出現在倒數計時器的位置。若要新增倒數計時器，請依照下列步驟執行：

  1. 點擊倒數計時設定頁面上的鏈接。
  2. 找到活動倒數部分，並添加直播活動的日期和時間。
  3. 點擊保存。
  4. 按一下倒數計時器上方的文字來自訂。
筆記：您（發布者）有責任更改主頁狀態。圖庫永遠不會自動切換主頁狀態。

設定行事曆提醒

您可以設定網站，讓使用者可以按一下連結以下載行事曆提醒。範例行事曆提醒如下所示。

點擊日曆鏈接將下載一個 .ics 文件，該文件是程序包括谷歌日曆，蘋果日曆和微軟 Outlook 使用的日曆文件的全局格式。請注意，除非已設定行事曆設定，否則不會顯示行事曆提醒。

若要設定行事曆提醒 .ics 檔案的設定，請依照下列步驟執行：

  1. 點擊倒數計時設定關聯。
  2. 設定行事曆提醒 .ics 檔案詳細資料。
  3. 點擊保存。

新增視訊

點擊+添加視頻鏈接以將視頻收藏添加到該站點。有關創建集合的信息，請參見將視頻添加到門戶體驗。

系列清單中的第一個系列將會顯示在活動前的首頁。要查看所有收藏集，請單擊主頁上的視頻之一，或配置視頻存檔鏈接對象，以便觀眾可以單擊它以查看所有集合。
