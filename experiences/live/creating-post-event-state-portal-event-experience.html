為門戶事件體驗創建事件後狀態

{{ page.description }}

實時事件門戶模板提供賽前，現場直播，事件後，視頻存檔和視頻詳細信息頁面模板。您可以自訂這些頁面範本，讓網站在現場活動之前、期間和之後有不同的外觀。事件後狀態是網站在即時事件完成後顯示的狀態。

若要建立網站的事件後首頁狀態，請建立新的即時活動入口網站體驗。在圖庫網站編輯器中，單擊頁數在左側導航中，然後單擊主頁：賽后。

事件後的首頁狀態提供下列內容區域。

  • A-主頁徽標和名稱 -用於將查看器返回首頁。這裡有兩個佔位符，所以你可以選擇有一個圖像後跟文本例如。
  • B-鏈接到視頻存檔 -用於查看已為該網站創建的所有視頻集。
  • C-Twitter設置 -提供使用Twitter feed小部件的功能。看配置Twitter提要。
  • D-背景圖片，文字和鏈接 -用於添加背景圖像，字幕文本和鏈接
  • E-直播活動名稱 -靜態短信
  • F -直播已結束消息 -靜態短信
  • G-電子郵件註冊設置 -用於配置電子郵件註冊。看配置電子郵件註冊。
  • H -新增影片 -用於向網站添加視頻收藏。看添加視頻想要查詢更多的信息。
  • 一世 -視頻網格盒 -用於添加號召性用語，包括自定義HTML或圖像。看在門戶網站體驗上配置號召性用語想要查詢更多的信息。
  • J-頁腳對象 -用於添加文本和頁面頁腳的鏈接

有關將圖像，鏈接和文本添加到“圖庫"頁面模板的信息，請參閱定制門戶網站體驗頁面的內容。

筆記：網站顏色將根據所選主題而有所不同。

設定推特動態消息

該網站可以配置在頁面上的 Twitter 嵌入。對於可用的嵌入類型，請檢查Twitter網站。一個樣本喜歡的時間軸如下所示。

若要新增 Twitter 動態消息，請依照下列步驟執行：

  1. 在Twitter站點上創建一個小部件。有關創建Twitter小部件的幫助，請參閱Twitter開發人員文檔。
  2. 點擊Twitter設置頁面上的鏈接。
  3. 複製 Twitter 小工具的 HTML，然後將其貼到 Twitter 動態消息 設定中。
  4. 點擊保存。

設定電子郵件註冊

您可以設定網站，讓網站訪客可以提供其電子郵件地址。

若要新增電子郵件註冊，請依照下列步驟執行：

  1. 點擊電子郵件註冊設置頁面上的鏈接。
  2. 視需要設定電子郵件註冊屬性。
    • 不顯示電子郵件潛在客戶表格 -不要顯示電子郵件表格
    • 使用模板隨附的電子郵件表單 -選擇以使用模板隨附的默認電子郵件表單
    • 使用您的營銷自動化軟件提供的自定義HTML -選擇以使用自定義表單HTML捕獲電子郵件
  3. 點擊保存。
  4. 配置完電子郵件設置後，返回頁面並單擊電子郵件註冊消息鏈接並設置顯示在電子郵件註冊字段上方的文本。

電子郵件表單捕獲的所有數據都可以使用出口畫廊線索受眾群體模塊中的選項。使用自訂 HTML 選項擷取的潛在客戶可在行銷自動化軟體中檢視。

新增視訊

點擊+添加視頻鏈接以將視頻收藏添加到該站點。有關創建集合的信息，請參見將視頻添加到門戶體驗。

系列清單中的第一個系列將會顯示在活動後的首頁。要查看所有收藏集，請單擊主頁上的視頻之一，或配置視頻存檔鏈接對象，以便觀眾可以單擊它以查看所有集合。
