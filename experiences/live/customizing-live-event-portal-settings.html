自定義實時事件門戶設置

{{ page.description }}

建立即時事件入口網站體驗時，可以設定數個範本特定設定。要配置Live Event Portal模板的設置，請編輯體驗並單擊外觀和行為>模板設置在左側導航欄中。

即時事件入口網站範本具有各種設定，可以針對每個首頁狀態進行設定。

按一下索引標籤以設定每個狀態的設定。這些設定會記錄在適當的頁面上的首頁狀態。如需相關資訊，請參閱下列其中一個主題：

  • 為門戶事件體驗創建事件前狀態
  • 為門戶事件體驗創建現場事件狀態
  • 為門戶事件體驗創建事件後狀態
