圖庫模板發行說明

{{ page.description }}

隨著時間的推移，Brightcove 可能會對範本進行新功能和增強功能。布萊特灣不會將重大範本更新推送到現有的網站。發行者負責將其網站升級為使用新範本，然後在發佈之前測試網站。有關更新圖庫網站以使用新模板版本的信息，請參見更新圖庫模板。

範本所做的變更會記錄為範本版本版本說明的一部分。版本說明適用於下列範本：

  • 目錄
  • 編年史
  • 經典的
  • 現場直播
  • 選框
  • 馬賽克
  • 發行人
  • 展示櫃

目錄發行說明

五季至二零一八年二月十四日

  • 新增選擇性標題至首頁

第 4.0 版至二零一八年一月二十四日

  • 當收藏只包含一個視頻時，更新了旋轉木馬在移動設備上將視頻置中
  • 使旋轉木馬過渡更清潔

編年史發行紀錄

第三季至二零一六年十二月二十二日

  • 使用新的影像載入器改善縮圖影像

第 2.0 版-二零一六年十月三十一日

  • 為英雄影片新增兩個版面配置選項
    • 新版面配置的主體圖片較小，適合沒有高品質縮圖影像的出版商
  • 刪除了分頁以支持滾動時延遲加載
  • 新增延遲載入影片的 CTA 支援
    • 類似於展示編年史可以選擇在 5,10 和 15 個視頻之後重複 CTA
  • 更新的集合導航/菜單佈局
    • 將全螢幕導覽取代為畫面左側的導覽列
