    布萊特灣畫廊™ 產品更新

    本主題概述了已對布萊特灣畫廊應用程式所做的更新。

    發行說明還可用於：

    • 圖庫範本版本發行資訊

    自動播放視訊設定

    9月4日  2019年

    一個新的自動播放影片設置已添加，可用於控制詳細信息頁面上視頻的播放行為。有關信息，請參見自定義播放器和潛在客戶表單設置以獲得門戶體驗。

    重新設計的圖庫導覽

    9月27日  2018年

    編輯入口網站體驗時的導覽功能表經過重新設計，使其與頁面內體驗編輯環境更直觀且一致。有關信息，請參見使用網站編輯器定制門戶體驗。

    搜尋選項新增至圖片庫首頁

    2018年8月1日

    一種搜索框已添加到“圖庫”主頁，提供了搜索體驗列表的功能。搜尋時，會搜尋下列屬性：

    • 標題
    • 描述
    • ID
    • 範本
    • 全球資源定位器

    將視訊新增至頁面內體驗

    七月十八日

    將影片新增至頁面內體驗時，現在可以選擇使用播放清單參考 ID。透過使用播放清單參考 ID，使用媒體模組將參考 ID 重新指定給其他播放清單，即可輕鬆變更體驗中的影片。當您需要變更體驗中顯示的影片時，不需 要編輯和重新發佈體驗。有關信息，請參見使用播放列表參考ID。

    潛在客戶表單的變更

    2018年5月15日

    「圖庫」現在使用「對象」模組來建立和管理潛在客戶表單。受眾將收集潛在客戶資料並提供資料匯出選項。有關信息，請參見使用潛在客戶表單進行門戶體驗。

    即時活動頁面內範本

    2018年5月14日

    圖庫現在提供即時活動頁面內範本，可用來傳遞即時活動作為頁面體驗的一部分。如需資訊，請參閱建立即時事件頁面體驗。

    入口網站體驗網站設定 UI 更新

    2018年4月13日

    網站設置用戶界面已更新，可以從自訂對話到新視頻設置對話框。有關更多信息，請參見定制視頻設置以獲得門戶體驗。

    圈選範本設定

    2018年3月30日

    「選取畫面」範本中新增了新的特色視訊行為設定。有關更多信息，請參見自定義選取框模板設置。

    自定義頁眉和頁腳選項

    2018年3月30日

    現在可以在使用自訂頁首和頁尾時包含預設頁首和頁尾。有關更多信息，請參見配置自定義頁眉/頁腳HTML。

    頁內體驗3Play組件

    二月一日

    3 Play 媒體互動字幕元件現在可以新增至頁面內體驗。這可讓觀眾在播放影片時看到影片字幕。有關更多信息，請參見將組件添加到頁內體驗。

    頁內體驗聊天組件

    2018年1月17日

    聊天元件現在可以新增至頁面內體驗。有關更多信息，請參見將組件添加到頁內體驗。

    定制的訪問控制頁面

    十一月二十八日

    存取控制頁面現在可以設定樣式，使其與網站的其他部分更加密切。有關更多信息，請參見設置訪問控制頁面的樣式。

    程式化404頁

    二零一七年十一月六日

    404 頁面現在將使用與網站配置的樣式（頁眉和頁腳），因此它們與網站的其他部分更密切的品牌。

    即時事件範本-背景影像選項

    二零一七年十一月六日

    即時事件範本現在提供了如何處理背景影像的選項。有關更多信息，請參見自定義實時事件門戶設置。

    集合中繼資料

    二零一七年十一月六日

    對於所有範本 (選取畫面除外)，您現在可以新增集合描述，並將做為類別頁面的中繼資料描述 (適用於共用該集合的瀏覽器)。有關更多信息，請參見將視頻雲視頻添加到門戶體驗。

    不再需要API密鑰

    九月二十九日

    API 金鑰不再需要，並已從 [設定] 頁面移除。

    同一頁面上的多種頁內體驗

    2017年8月15日

    資源庫現在支援在同一頁面上擁有相同頁面內體驗的多個執行個體。這可讓您建立一個體驗，然後動態傳遞播放清單或要顯示的影片清單。有關更多信息，請參見在頁內體驗中使用動態視頻替代。

    事件體驗現在支持使用參考ID

    8月9   2017年

    設定事件體驗的即時事件狀態時，現在可以使用參考 ID 來指定即時視訊資產。使用參考 ID 可提供更大的彈性，因為參考 ID 可以指派給即時視訊資產，而無需變更體驗。有關更多信息，請參見創建活動體驗的現場活動狀態。

    其他頁內體驗播放器設置

    7月27   2017年

    新增播放器設定，讓頁內體驗在影片完成時保持播放狀態。這可以用來顯示播放器端螢幕。有關更多信息，請參見配置頁內體驗的設置。

    頁面內體驗中的動態視訊覆寫

    7月27   2017年

    現在可以藉由修改內嵌程式碼，動態覆寫與頁內體驗相關聯的視訊。有關更多信息，請參見在頁內體驗中使用動態視頻替代。

    頁內體驗培訓現已推出

    7月21   2017年

    建立頁面內體驗的隨選訓練現已推出。有關更多信息，請參見按需培訓：使用圖庫建立頁內體驗。

    YouTube視頻支持頁內體驗

    7月11   2017年

    現在您可以在頁面內體驗中使用 YouTube 影片。有關更多信息，請參見將視頻添加到頁內體驗。

    在搜索引擎結果中使用網站描述

    7月10   2017年

    「圖庫」現在提供了一個在搜尋引擎結果中使用網站說明的選項。這將使用說明 標籤中的網站描述，包括 Facebook 和 Twitter 中繼標記。有關更多信息，請參見配置門戶網站體驗的搜索引擎優化設置。

    目錄模板

    2017年6月28日

    圖庫現在提供目錄範本。有關更多信息，請參見圖庫門戶網站模板概述。

    頁內體驗

    五月三十一日

    資源庫現在提供建立頁面內體驗的選項。有關更多信息，請參見概述：建立、編輯和發佈頁面內體驗。
