Brightcove Gallery入門

{{ page.description }}

如果您想要快速開始建立和發佈「圖庫」體驗，請查看下列其中一個主題：

  • 門戶體驗 -- 一步步：創建和發布門戶體驗
  • 頁內體驗 -- 一步步：創建和發布頁內體驗
  • 現場活動門戶 -- 創建現場活動門戶體驗
  • 直播活動頁內 -- 創建現場活動頁內體驗
  • 登陸頁面 -- 創建登錄頁面體驗

有關畫廊的概述並了解可以創建的體驗類型，請參加按需培訓：畫廊簡介。

 

使用布萊特灣畫廊

要訪問圖庫模塊，請登錄到Video Cloud Studio，然後單擊畫廊在導航菜單中。

您可以在首頁：

  • 創建和管理體驗
  • 配置圖庫設置

在創建門戶體驗之前，您應該配置圖庫設置。

建立和管理體驗

打開圖庫模塊時，體會默認情況下會顯示頁面。如果尚未建立任何體驗，系統會提示您建立新的體驗。有關可以創建的體驗類型的概述，請參見畫廊體驗概述。

要了解有關創建和管理Gallery體驗的更多信息，請參閱創建和管理畫廊體驗。

設定資源庫設定

創建門戶網站體驗時，必須為Gallery配置 默認播放器。要了解有關圖庫設置的更多信息，請參見配置圖庫設置。
