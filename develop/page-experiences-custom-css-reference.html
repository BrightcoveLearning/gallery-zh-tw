頁面內容

    頁面內體驗自訂 CSS 參考

    本主題概述建立頁面內體驗時所使用的 CSS 類別。提供此資訊，讓您可以使用 CSS 進一步設計您的體驗。

    樣式化元件

    這些 CSS 類適用於通過 UI 樣式的項目。 通過使用這些類，可以使用自定義CSS覆蓋這些樣式。 您還可以使用這些類將UI樣式應用於自定義HTML。 這些類是穩定的，不應該在模板版本之間更改。

     

    類別 描述
    電子組件-樣式-全局 體驗的全球造型
    電子組件-樣式錨點 一般連結樣式
    電子組件-樣式導航 導航鏈接樣式（後退按鈕，共享按鈕等）
    電子元件樣式視訊資訊 視訊資訊方塊樣式 (非覆疊)
    E-組件-樣式-視頻標題 視訊標題樣式 (非覆蓋)
    E-元件樣式-視訊說明 視訊描述樣式 (非覆疊)
    電子組件-樣式覆蓋 覆疊視訊資訊方塊樣式
    E-組件-樣式-重疊標題 覆疊視訊標題樣式
    E-元件-樣式-覆疊描述 覆疊視訊描述樣式
    E-組件-樣式-覆蓋橫幅 覆蓋橫幅/指示器樣式（觀看指示器）
    E-組件-風格-按鈕 播放按鈕基本樣式（請參閱播放按鈕組件以獲取更複雜的樣式）

    基本元件

    這些是可以使用自定義 CSS 進行樣式化的基本組件。 組件類是穩定的，不應在模板版本之間進行更改。 內部類相對穩定，並且很可能在模板版本之間不會更改。

    播放按鈕

    播放按鈕會出現在每個視訊縮圖上。

     

    類別 描述
    E-組件-播放按鈕 播放按鈕組件
    E-組件-播放按鈕-svg SVG 容器
    E-組件-播放按鈕組 SVG 內部群組
    E-組件-播放按鈕 播放按鈕 SVG 群組
    電子組件-播放按鈕框架 播放按鈕圓框
    E-組件-播放按鈕圖標 播放按鈕圖示
    E-組件-播放按鈕倒數計時 倒數 SVG 群組
    E-組件-播放按鈕-暫停 倒數暫停圖示
    E-組件-播放按鈕文本 倒數計時文字
    E-組件-播放按鈕軌道 倒數曲目
    E-組件-播放按鈕-亞軍 倒數進度

     

    箭頭

    旋轉木馬範本的箭頭 (適用於單一和完整出血的簡報選項)。

     

    類別 描述
    電子元件箭頭 箭頭元件 (包括背景)
    E-組件-箭頭-內部 箭頭

     

    播放下一個

    在以下情況下，大縮略圖上會顯示上一個下一個指示符自動前進到下一個視頻在播放器設置中啟用。

     

    類別 描述
    e-組件-下一個 上下一個元件
    元件-下一個縮圖 下一個視訊縮圖
    元件-下一個文字 向上下一個文字 (倒數 + 影片名稱)
    元件-下一個文字倒數計時 倒數計時文字
    e-組件-下一個近距離 關閉按鈕

     

    觀看的視訊指示器

    在「視訊」設定中啟用此選項時，會顯示「觀看指示器」。

     

    類別 描述
    電子元件觀看影片橫幅 觀看的視訊指示器

    共用元件

    與社交分享相關的元件。 共享按鈕類相對穩定，在模板版本之間很可能不會更改。 共享面板的類別很可能在不久的將來發生變化，應視為不穩定。

    「共享」按鈕

    「分享」按鈕或圖示。

    或

     

    類別 描述
    電子組件共享按鈕 共用按鈕元件
    E-組件共享按鈕按鈕 「分享」按鈕 (非圖示)

     

    共用面板

    社交分享顯示設定為按鈕時的分享面板。

    注意：這些課程將來可能會有所變更。

     

    類別 描述
    e-組件共享包裝面板 面板包裝元件
    e-組件共享-包裝-內部 面板
    e-組件-共享-包裝-關閉 關閉按鈕
    e-組件共享-包裝項 共用項目
    e-組件共享-包裝-項目名稱 共用項目名稱

     

    可編輯的元件
