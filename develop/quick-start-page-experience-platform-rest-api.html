快速開始：頁內體驗平台（REST）API

快速入門提供了一些用於管理頁面內體驗的 API 請求範例，並提供使用這些要求的範例介面。

簡介

頁面內體驗 API 可讓您以程式設計方式管理頁面內體驗，使其對 CMS 整合特別有用，允許使用者從自己的 CMS 建立管理體驗，而不是前往 Studio 中的圖庫。

此快速入門提供範例實作，可讓使用者：

  1. 取得帳戶的所有頁面內體驗範本
  2. 從選取的範本建立基本體驗
  3. 選擇播放清單或影片清單以加入體驗
  4. 更新選取的影片/播放列表的體驗
  5. 發佈體驗
